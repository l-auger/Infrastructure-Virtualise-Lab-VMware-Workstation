# Application des stratégies de groupe (GPO)

Les stratégies de groupe permettent de centraliser la configuration et la sécurité des postes clients.

Sur ce poste Windows 11, les GPO sont fournies par les contrôleurs de domaine.

## Vérifications effectuées
- Application des GPO machine et utilisateur
- Communication correcte avec les DC
- Rafraîchissement automatique des stratégies

## Commandes utilisées
- gpresult /r
- rsop.msc
