# Debian 12 – Serveur Linux

Ce dossier documente l’utilisation d’un serveur **Debian 12** au sein de l’infrastructure.

Il permet d’introduire un environnement Linux afin de diversifier les systèmes présents dans le laboratoire.

## Rôle du serveur

- Serveur Linux générique
- Support de tests et d’apprentissage
- Hébergement de services légers

## Objectifs

- Manipuler un système Linux serveur
- Comprendre l’intégration dans un réseau Active Directory
- Appliquer les bonnes pratiques de base (réseau, services, sécurité)

Ce serveur sert de base évolutive pour de futurs services.
